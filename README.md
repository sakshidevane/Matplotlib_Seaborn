# Matplotlib_Seaborn

# Data Visualization (Matplotlib & Seaborn) 📈

## Overview

This repository contains my **Data Visualization notebook** built using **Matplotlib and Seaborn** as part of my Python learning journey. The notebook focuses on creating clear, meaningful, and visually appealing plots to understand and communicate data effectively.

The aim of this project is to move beyond raw data and learn how to **present insights visually**, which is an essential skill in data analysis and data science.

## Project Objective

* Learn and apply **data visualization techniques**
* Understand how to represent data using different types of plots

## How to Run the Notebook

1. Clone this repository
2. Install required libraries:

   ```bash
   pip install matplotlib seaborn jupyter
   ```
3. Open Jupyter Notebook
4. Run `Matplotlib_Seaborn.ipynb`
